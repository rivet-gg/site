# Introduction

Find documentation for our APIs, detailed concept rundowns, or get help with specific engine plugins.

## What does Rivet provide?

- [Dynamic Servers](/docs/general/dynamic-servers/overview) for auto-scaling game lobbies
- [Game Guard](/docs/general/dynamic-servers/concepts/game-guard) for DDoS mitigation and managed WebSocket SSL & TCP+TLS termination
- Streamlined DevOps for teams
- Unified logging & monitoring & analytics
- No downtime deploys with easy rollbacks

## Low-Level API
