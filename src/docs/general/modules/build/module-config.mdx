# Config (module.json)

## Module

_Object containing the following properties:_

| Property          | Description                                                                | Type                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| :---------------- | :------------------------------------------------------------------------- | :----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `status`          | The status of the module.                                                  | `'coming_soon' \| 'preview' \| 'beta' \| 'stable' \| 'maintenance' \| 'end_of_life'`                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| `name`            | The human readable name of the module.                                     | `string`                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
| `description`     | A short description of the module.                                         | `string`                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
| `icon`            | The [Font Awesome](https://fontawesome.com/icons) icon name of the module. | `string`                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
| `tags`            | The tags associated with this module.                                      | `Array<string>`                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
| `authors`         | The GitHub handle of the authors of the module.                            | `Array<string>`                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
| `scripts`         | The scripts associated with this module.                                   | _Object with dynamic keys of type_ `string` _and values of type_ _Object with properties:_<ul><li>`name`: `string` - The human readable name of the script.</li><li>`description`: `string` - A short description of the script.</li><li>`public`: `boolean` - If the script can be called from the public HTTP interface.</li></ul>                                                                                                                                                                                                                                                                                                                         |
| `actors`          | The actors associated with this module.                                    | _Object with dynamic keys of type_ `string` _and values of type_ _Object with properties:_<ul><li>`storageAlias`: `string` - Used to keep actor IDs the same in case the actor name changes.</li></ul>                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
| `routes`          | The routes associated with this module.                                    | _Object with dynamic keys of type_ `string` _and values of type_ _Object with properties:_<ul><li>`name`: `string` - The human readable name of the route.</li><li>`description`: `string` - A short description of the route.</li><li>`method`: `string` - The HTTP method of the route.</li><li>`path`: `string` - The path of the route.</li></ul> _or_ _Object with properties:_<ul><li>`name`: `string` - The human readable name of the route.</li><li>`description`: `string` - A short description of the route.</li><li>`method`: `string` - The HTTP method of the route.</li><li>`pathPrefix`: `string` - The path prefix of the route.</li></ul> |
| **`errors`** (\*) | The errors associated with this module.                                    | _Object with dynamic keys of type_ `string` _and values of type_ _Object with properties:_<ul><li>`name`: `string` - The human readable name of the error.</li><li>`description`: `string` - A short description of the error.</li><li>`internal`: `boolean` - If the error should only be shown internally.</li></ul>                                                                                                                                                                                                                                                                                                                                       |
| `dependencies`    | The dependencies of this module.                                           | _Object with dynamic keys of type_ `string` _and values of type_ _Object with properties:_<ul></ul>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
| `defaultConfig`   | Default user config.                                                       | `unknown` (_nullable_)                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |

_(\*) Required._
