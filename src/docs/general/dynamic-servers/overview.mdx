# Dynamic Servers

<Summary>Auto-scaling game lobbies boot up to meet demand.</Summary>

Dynamic Servers is the system that automatically operates and scales your game servers.

## Getting Started

To get started with Dynamic Servers:

1. [Create a new game token](/docs/core/game-tokens/api/create-service-token)
2. [Upload a game build](/docs/core/dynamic-servers/api/builds/prepare) (and [complete the upload](/docs/core/dynamic-servers/api/builds/complete))
3. [Create a server](/docs/core/dynamic-servers/api/create)
