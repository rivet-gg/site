## Add development token to config

We will configure the token to connect to Rivet to test the game.

1. Create a development token by running the following command and copying the token. It should look like `dev_staging.XXXX`.

   ```bash
   dev create-dev-token --format json
   ```

2. Add the following to the end of _Config/DefaultGame.ini_. Replace `dev_staging.XXXX` with the token created in the last step.
