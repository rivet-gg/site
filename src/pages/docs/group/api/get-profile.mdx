# getProfile

## Description

Returns a group profile.

## Response Body

### `group`

_object, required_

A list of group profiles.

### `group.avatar_url`

_string_

The URL of this group's avatar image.

### `group.bio`

_string_

Detailed information about a profile.

### `group.display_name`

_string_

Represent a resource's readable display name.

### `group.external`

_object_

External links for this group.

### `group.external.profile`

_string_

A link to this group's profile page.

### `group.group_id`

_string_

### `group.is_current_identity_member`

_boolean_

Whether or not the current identity is a member of this group.

### `group.is_current_identity_requesting_join`

_boolean_

Whether or not the current identity is currently requesting to join this group.

### `group.is_developer`

_boolean_

Whether or not this group is a developer.

### `group.join_requests`

_array_
