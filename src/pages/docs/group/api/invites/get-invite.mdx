# invites.getInvite

## Description

Inspects a group invite returning information about the team that created it.


## Code Examples

<CodeGroup title='Request' tag='GET' label='https://api.rivet.gg/group/invites/{group_invite_code}'>

```bash {{ "title": "cURL" }}
curl -X GET 'https://api.rivet.gg/group/invites/{group_invite_code}'
```

```ts
// Create Rivet client
import { RivetClient } from '@rivet-gg/api';
const RIVET = new RivetClient({ token: addYourTokenHere });

// Make request
await RIVET.group.invites.getInvite({
  // Add your request body here
});
```

</CodeGroup>


## Request Parameters


### `group_invite_code`

_Path parameter, required_


Provided by `rivet.api.group#CreateInviteResponse$code`.


## Response Body

### `group`

`object` (required)

A group handle.

### `group.avatar_url`

`string`

The URL of this group's avatar image

### `group.display_name`

`string` (required)

### `group.external`

`object` (required)

External links for this group.

### `group.external.profile`

`string` (required)

A link to this group's profile page.

### `group.group_id`

`string` (required)

### `group.is_developer`

`boolean`

Whether or not this group is a developer group.
