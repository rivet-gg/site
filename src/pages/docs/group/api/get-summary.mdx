## Response Body

### `group`

`object` (required)

### `group.avatar_url`

`string`

The URL of this group's avatar image.

### `group.bio`

`string` (required)

Follows regex ^(?:[^\n\r]+\n?|\n){1,5}$

### `group.display_name`

`string` (required)

### `group.external`

`object` (required)

External links for this group.

### `group.external.profile`

`string` (required)

A link to this group's profile page.

### `group.group_id`

`string` (required)

### `group.is_current_identity_member`

`boolean` (required)

Whether or not the current identity is a member of this group.

### `group.is_developer`

`boolean` (required)

Whether or not this group is a developer.

### `group.member_count`

`integer` (required)

### `group.owner_identity_id`

`string` (required)

### `group.publicity`

`string` (required)

The current publicity value for the given group.
