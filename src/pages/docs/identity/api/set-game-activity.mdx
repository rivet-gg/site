# setGameActivity

## Description

Sets the current identity's game activity. This activity will automatically be removed when the identity goes offline.

## Request Parameters


## Request Body

### `game_activity`

`object` (required)

Information about the identity's current game. This is information that all other identities can see about what the current identity is doing.

### `game_activity.message`

`string`

A short message about the current game activity.

### `game_activity.mutual_metadata`

`object`

JSON data seen only by the given identity and their mutual followers.

### `game_activity.public_metadata`

`object`

JSON data seen by anyone.




## Response Body

_Empty response body._
