# getProfile

## Description

Fetches an identity profile.


## Code Example

<CodeGroup title='Request' tag='GET' label='https://identity.api.rivet.gg/v1/identities/{identity_id}/profile'>

```bash {{ title: 'cURL' }}
curl -X GET 'https://identity.api.rivet.gg/v1/identities/{identity_id}/profile'
```

```ts
// Create Rivet client
import { RivetClient } from '@rivet-gg/api';
const RIVET = new RivetClient({ token: addYourTokenHere });

// Make request
await RIVET.identity.getProfile({
  // Add your request body here
});
```

</CodeGroup>
