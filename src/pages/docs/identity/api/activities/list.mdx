# activities.list

## Description

Returns an overview of all players currently online or in game.


## Code Examples

<CodeGroup title='Request' tag='GET' label='https://api.rivet.gg/identity/activities'>


```ts
// Create Rivet client
import { RivetClient } from '@rivet-gg/api';

// Make request

  // Add your request body here
});
```
