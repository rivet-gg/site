# Dynamic Servers

<Summary>Auto-scaling game lobbies boot up to meet demand.</Summary>

Dynamic Servers is the system that automatically operates and scales your game servers.

There is no API for Dynamic Servers. Lobbies are created on demand when [matchmaker.lobbies.find](/docs/matchmaker/api/lobbies/find) is called. Read more about the matchmaker [here](/docs/matchmaker/index).
