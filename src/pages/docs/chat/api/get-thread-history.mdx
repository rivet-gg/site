# getThreadHistory

## Description

Returns message history for a given thread in a certain direction.
Defaults to querying messages before ts.


## Code Examples

<CodeGroup title='Request' tag='GET' label='https://chat.api.rivet.gg/v1/threads/{thread_id}/history'>

```bash {{ title: 'cURL' }}
curl -X GET 'https://chat.api.rivet.gg/v1/threads/{thread_id}/history'
```

```ts
// Create Rivet client
import { RivetClient } from '@rivet-gg/api';
const RIVET = new RivetClient({ token: addYourTokenHere });

// Make request
await RIVET.chat.getThreadHistory({
  // Add your request body here
});
```

</CodeGroup>


## Request Parameters


### thread_id

_Path parameter, required_





### ts

_Query parameter, optional_


RFC3339 timestamp.


### count

_Query parameter, required_


How many messages to collect in each direction. If querying `rivet.api.chat.common#QueryDirection$before_and_after`, `rivet.api.chat.common#QueryDirection$chat_messages` will be `count * 2`.


### query_direction

_Query parameter, optional_


Represents which direction to query messages from relative to the given timestamp.
