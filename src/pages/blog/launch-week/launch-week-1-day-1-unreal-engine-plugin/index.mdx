The official Rivet Unreal Engine module allows you to deploy your multiplayer game to Rivet.

‍

Integrating Rivet with Unreal Engine just requires you to call the Rivet Matchmaker find lobby function and build a Linux dedicated server. Once you've done that, run *rivet deploy* and we'll take it from there.

‍
