# rivet.game Domain

## Overview

Rivet provides you a domain to hose your assets on `{game}.rivet.game` (e.g. `my-game.rivet.game`).

You can find the domain by clicking _Visit Site_ on the Rivet dashboard.

## Namespacing

Each namespace for your game has a dedicated subdomain. This allows you to quickly test standalone variations of your assets.

The namespace sites can be accessed at `{game}--{namespace}.rivet.game`. For example, if your game is named `my-game`,you can access the `staging` namespace at `my-game--staging.rivet.game`.

The `prod` namespace can be used without `--`. For example, `my-game.rivet.game` is equivalent to `my-game--prod.rivet.game`.

<Tip>Make sure you use two dashes before the namespace (`--`).</Tip>

This link is also accessible by clicking _Visit Site_ on the Rivet dashboard.
