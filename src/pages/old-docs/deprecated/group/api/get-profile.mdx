# getProfile

## Description

Returns a group profile.

## Code Examples

<CodeGroup title='Request' tag='GET' label='https://api.rivet.gg/group/groups/{group_id}/profile'>

```bash {{ "title": "cURL" }}
curl -X GET 'https://api.rivet.gg/group/groups/{group_id}/profile'
```

```ts
// Create Rivet client
import { RivetClient } from '@rivet-gg/api';

// Make request

  // Add your request body here
});
```

### `group_id`

_Path parameter, required_

### `watch_index`

_Query parameter, optional_

A query parameter denoting the requests watch index.
