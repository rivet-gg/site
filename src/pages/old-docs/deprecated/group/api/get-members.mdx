# getMembers

## Description

Returns a group's members.

## Code Examples

<CodeGroup title='Request' tag='GET' label='https://api.rivet.gg/group/groups/{group_id}/members'>

```bash {{ "title": "cURL" }}
curl -X GET 'https://api.rivet.gg/group/groups/{group_id}/members'
```

```ts
// Create Rivet client
import { RivetClient } from '@rivet-gg/api';
const RIVET = new RivetClient({ token: '[YOUR TOKEN HERE]' });

// Make request
await RIVET.group.getMembers({
  // Add your request body here
});
```

</CodeGroup>

## Request Parameters

### `group_id`

_Path parameter, required_

### `anchor`

_Query parameter, optional_

The pagination anchor. Set to the returned anchor of this endpoint to receive the next set of items.

### `count`

_Query parameter, optional_

Amount of members to return.

### `watch_index`

_Query parameter, optional_

A query parameter denoting the requests watch index.

## Response Body

### `anchor`

`string`

The pagination anchor.

### `members`

`array<object>` (required)

A list of group members.

### `members[*].identity`

`object` (required)

An identity handle.

### `members[*].identity.account_number`

`integer` (required)

### `members[*].identity.avatar_url`

`string` (required)

The URL of this identity's avatar image.

### `members[*].identity.display_name`

`string` (required)

### `members[*].identity.external`

`object` (required)

External links for an identity.

### `members[*].identity.external.profile`

`string` (required)

A link to this identity's profile page.

### `members[*].identity.external.settings`

`string`

A link to the Rivet settings page.

### `members[*].identity.identity_id`

`string` (required)

### `members[*].identity.is_registered`

`boolean` (required)

Whether or not this identity is registered with a linked account.

### `members[*].identity.presence`

`object`

Information about the identity's current status, party, and active game.

### `members[*].identity.presence.game_activity`

`object`

The game an identity is currently participating in.

### `members[*].identity.presence.game_activity.game`

`object` (required)

### `members[*].identity.presence.game_activity.game.banner_url`

`string`

The URL of this game's banner image.

### `members[*].identity.presence.game_activity.game.display_name`

`string` (required)

### `members[*].identity.presence.game_activity.game.game_id`

`string` (required)

### `members[*].identity.presence.game_activity.game.logo_url`

`string`

The URL of this game's logo image.

### `members[*].identity.presence.game_activity.game.name_id`

`string` (required)

A human readable short identifier used to references resources. Different than a `uuid` because this is intended to be human readable. Different than `DisplayName` because this should not include special characters and be short.

### `members[*].identity.presence.game_activity.message`

`string` (required)

A short activity message about the current game activity.

### `members[*].identity.presence.game_activity.mutual_metadata`

`object`

JSON data seen only by the given identity and their mutual followers.

### `members[*].identity.presence.game_activity.public_metadata`

`object`

JSON data seen by anyone.

### `members[*].identity.presence.status`

`string` (required)

The current status of an identity. This helps players understand if another player is currently playing or has their game in the background.

### `members[*].identity.presence.update_ts`

`string` (required)

### `watch`

`object` (required)

Provided by watchable endpoints used in blocking loops.

### `watch.index`

`string` (required)

Index indicating the version of the data responded.
Pass this to `WatchQuery` to block and wait for the next response.
