# invites.createInvite

## Description

Creates a group invite. Can be shared with other identities to let them join this group.

## Request Parameters

### `group_id`

_Path parameter, required_

## Request Body

### `ttl`

`number`

How long until the group invite expires (in milliseconds).

### `use_count`

`number`

How many times the group invite can be used.

## Response Body

### `code`

`string` (required)

The code that will be passed to `rivet.api.group#ConsumeInvite` to join a group.
