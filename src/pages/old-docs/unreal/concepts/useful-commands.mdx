# Useful Unreal console commands for multiplayer

Unreal console commands can be run from the editor by pressing <code>`</code>.

<AccordionGroup>
	<Accordion title="ServerTravel">
        Used on the Unreal server to move the server to a new level.

    	[Related documentation](https://docs.unrealengine.com/5.2/en-US/API/Runtime/Engine/Engine/UWorld/ServerTravel/)
    </Accordion>
    <Accordion title="Open">
        Normally used to open a level, but it can also be used to connect to a server manually.
    </Accordion>
</AccordionGroup>
